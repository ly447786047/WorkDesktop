# 西部工作台
## 项目简介
本项目是面向新疆西部计划志愿者的轻量化本地工作台单页网站，采用原生 HTML / CSS / JavaScript 开发，无前端框架依赖，无需服务器，本地浏览器直接打开即可使用。
整合时钟、天气预警、多日预报、每日一言、连贯式上下班倒计时进度条、功能分区卡片、自动轮换背景图等功能，适配基层驻村、材料归档、工作记录、日常待办等办公场景。

## 功能介绍
### 1. 实时时钟
页面顶部常驻完整日期时间，每秒自动刷新，显示年、月、日、时、分、秒。

### 2. 天气模块
- 自动拉取泽普县实时天气，展示天气状况、实时气温、昼夜温差、空气湿度、紫外线等级；
- 鼠标悬浮天气区域，弹出多日天气预报卡片；
- 存在气象预警时自动弹出对应等级提示弹窗，区分红/橙/黄预警样式。

### 3. 每日一言
调用在线语录接口展示文案，定时自动更新；网络请求失败时自动展示西部计划专属备用文案。

### 4. 连贯式上下班倒计时进度条
自动根据当前月份切换喀什本地冬令时、夏令时两套作息，全天工作时长统一计算，午休不会重置进度，仅更换提示文字。
- 周末：提示休息，进度条归零；
- 凌晨至上班前1小时：休息提示；
- 上班前1小时：上班倒计时提醒；
- 上午在岗：进度持续上涨，显示距离午休剩余时长；
- 午休时段：进度保持上午完成占比，提示用餐休息；
- 下午在岗：在上午进度基础上继续递增，显示距离下班剩余时长；
- 下班前1小时：下班倒计时提醒；
- 下班至当日23:59：已下班提示，进度条拉满至100%。

### 5. 分区功能卡片
采用网格布局划分6个业务模块，磨砂玻璃渐变视觉风格：
- 常用软件
- 工作总文件夹
- 正在进行的工作
- 每日一言（含倒计时进度条）
- 西部计划日记 & 工作总结
- 今日待办工作

### 6. 自动轮换背景
定时随机读取 `images/background` 内图片作为页面背景，图片统一命名为 `bg_数字.jpg`；图片加载失败自动切换默认兜底背景图。

## 运行说明
1. 将整套文件解压至同一文件夹，目录结构不可缺失、改名；
2. 各类图片素材放入对应 `images` 子文件夹，背景图存放至 `images/background`；
3. 双击 `index.html`，使用任意现代浏览器打开即可正常运行。

## 批量重命名工具使用说明
`rename_bg.bat` 用于规范化批量重命名背景图片，解决文件名含特殊字符、重复冲突问题。
1. 将脚本放入 `images` 文件夹内；
2. 双击运行，自动扫描 `background` 目录下全部图片；
3. 采用临时中转命名机制，避免重复命名报错，最终统一输出 `bg_1.jpg`、`bg_2.jpg` 格式；
4. 运行结束仅输出处理成功、失败总数，无多余中间日志。

## 作息时间标准
### 夏令时（5月1日 — 9月30日）
- 上午工作：10:00 - 14:00
- 午休时段：14:00 - 16:00
- 下午工作：16:00 - 20:00

### 冬令时（10月1日 — 次年4月30日）
- 上午工作：10:00 - 14:00
- 午休时段：14:00 - 15:30
- 下午工作：15:30 - 19:30
